# Meizikt

> Gank.io Android客户端 使用Kotlin + Retrofit2 + Rx

### 数据来源：

  本应用所有数据均来自[干货集中营](http://gank.io/)的开放API。
  
### 应用简介：
  Meizikt，随时随地了解最新技(piao)术(liang)动(mei)态(zi)。除周末部分节假日外，每日更新一个美图，一个小视频，以及各类技术最新文章。
